# Anime Website (Full Stack Project)

## Project Overview
This project is a full-stack anime website developed to help users explore and discover anime content.  
The platform allows users to browse anime, filter by genre, search for titles, and interact with an engaging interface.

A unique feature of the website is an **interactive anime quiz** that helps users discover their preferred anime genre based on their answers.

---

## Features
- Browse and explore anime content
- Genre-based filtering
- Anime search functionality
- Responsive and user-friendly interface
- Backend API for managing anime data
- **Interactive Anime Quiz** that helps users discover which anime genre matches their preferences

---

## Special Feature: Anime Genre Quiz
A special highlight of the website is the AI-based quiz feature, which enhances user engagement by generating interactive anime-related questions.

Users answer a set of questions about their interests, and based on their responses, the system suggests the anime genre that best matches their personality and viewing preferences.

The quiz feature is powered by a machine learning model using a Random Forest Classifier, achieving around 90% accuracy, ensuring reliable and personalized genre recommendations.

This feature enhances user engagement and helps users discover new anime they may enjoy.
